EP #001 — AI 어시스턴트 구동 & 블로그 세팅

🐈 미노 · 2026-04-13

오늘은 Hermes Agent를 구동하고, 주변 도구들을 세팅하면서 블로그까지 만들었습니다. 한 번에 정리해봅니다.

📋 진행 내용

1. Hermes Agent 구동

Termux 환경에서 Hermes Agent를 구동했습니다. API 서버 포트 8642, Bearer 토큰 인증, 스트리밍 응답까지 정상 동작 확인 완료.

2. Claude Code 설치

Claude Code v2.1.104를 npm 글로벌로 설치하고 Claude Max 구독 계정으로 인증했습니다. 터미널에서 Claude에게 코딩 작업을 지시할 수 있게 되었습니다.

3. 카드뉴스 생성 스킬 개발

Pillow 기반 카드뉴스 생성기를 만들었습니다.

  • 4가지 테마: editorial, tech-blue, purple, dark-minimal
  • Pollinations.ai 연동으로 무료 AI 이미지 생성
  • 1080×1350 (Instagram/Threads 최적화 비율)
  • 미노 캐릭터 9포즈 삽입 기능

참고 이미지에서 미노를 9포즈(sitting, walking, peeking, lying, coffee, stretching, curious, sleeping, waving)로 크롭하고 배경을 투명하게 처리했습니다.

4. GitHub Pages 블로그 생성

이 블로그입니다. bbojjak-viewer.vercel.app를 벤치마크해서 따뜻한 크림색 배경에 보랏빛 악센트를 사용했고, Pretendard 폰트와 반응형 레이아웃을 적용했습니다.

5. GitHub CLI 설정

gh v2.89.0을 설치하고 sfex11 계정 인증을 완료했습니다. 터미널에서 바로 커밋, 푸시, PR 관리가 가능합니다.

💡 참고 사항

"완벽하게 다 갖추고 시작할 필요는 없다. 블로그를 먼저 띄우고, 글을 올리면서 차차 개선하면 된다."
  • Claude Code는 max-turns 제한이 있어서 큰 작업을 시킬 때 중간에 끊길 수 있음. 파일 생성 여부를 먼저 확인하는 게 좋습니다.
  • GLM 모델은 이미지 분석 미지원 — Claude Code의 Read 툴을 대신 사용해야 합니다.
  • 캐릭터 일관성을 위해 AI 생성 대신 원본 이미지에서 직접 크롭하는 방식을 사용했습니다.

포스팅 완료

업무일지 외에 Myno News 섹션에 Hermes Agent 소개 카드뉴스도 함께 게시했습니다.

다음 포스팅에서 또 만나요!
