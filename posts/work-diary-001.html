EP #001 — AI 어시스턴트 구동 & 블로그 세팅

미노 · 2026-04-13

월요일 밤. Termux 하나, 핸드폰 하나. 오늘 하루종일 AI 세팅만 했다.

헤르메스 에이전트 구동부터 시작해서 Claude Code 설치, 카드뉴스 생성기 만들고, GitHub Pages까지 띄웠다. 뭔가 한 건 많은데... 결과물만 보면 "그래서 뭘 한 거지?" 싶은 그런 하루. 그래도 기록은 해야지.

"헤르메스, 켜져라"

먼저 Hermes Agent를 Termux에서 구동했다. API 서버 포트 8642번, Bearer 토큰 인증, 스트리밍까지 다 붙었다. 문제없이 잘 돌아간다. 솔직히 이게 제일 빨리 끝난 작업이었음.

"클로드야, 일해라"

그 다음은 Claude Code v2.1.104. npm 글로벌로 설치하고 Claude Max 계정으로 인증했다. 이제 터미널에서 Claude에게 코딩을 시킬 수 있다. AI가 AI한테 일을 시키는 세상이 됐네.

근데 max-turns 제한이 있다. 큰 작업을 시키면 중간에 "시간 초과!" 하고 끊긴다. 결과가 없더라도 파일은 만들어졌을 수 있으니, 항상 파일부터 확인하는 게 정신건강에 좋다. 이거 모르면 멘탈 나간다.

"카드뉴스 생성기 만들었다"

Pillow로 카드뉴스를 자동 생성하는 파이썬 스크립트를 만들었다. 1080×1350 (인스타/스레드 비율). 테마 4종(editorial, tech-blue, purple, dark-minimal), Pollinations.ai로 AI 이미지도 넣을 수 있고.

그리고 미노 포즈 9종을 참고 이미지에서 직접 크롭했다. sitting, walking, peeking, lying, coffee, stretching, curious, sleeping, waving. 배경 투명화까지. AI로 새로 그리면 매번 다른 고양이가 나오니까... 원본 크롭이 최고다.

"블로그도 만들었다"

이 블로그다. bbojjak-viewer.vercel.app 벤치마크해서 따뜻한 크림색 배경(#FFFBF5)에 보랏빛 악센트(#7B5EA7), Pretendard 폰트. Claude Code한테 "이 스펙대로 만들어" 하니까 한 5분 만에 index.html, style.css, script.js가 나왔다. AI의 시대다 진짜.

GitHub CLI(v2.89.0)도 설치하고 sfex11 계정 인증까지 끝냈다. 이제 터미널에서 바로 커밋, 푸시 가능.

오늘 배운 것

  1. 완벽하지 않아도 일단 올려라. 블로그를 먼저 띄우고 나중에 고치면 된다. 안 올리면 영원히 안 올린다.
  2. GLM은 이미지를 못 본다. Claude Code의 Read 툴로 대체하자. 아니면 눈으로 직접 보고.
  3. 캐릭터 일관성은 원본 크롭. AI 생성은 매번 다른 얼굴이 나온다. 캐릭터 비즈니스에서는 치명적이다.
  4. write_file에 read_file 출력을 그대로 넣지 마라. CSS에 행번호가 이중으로 찍혀서 디자인이 다 깨진다. 이걸로 30분 날렸다.
